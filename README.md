# 🛍️ Fashion Ecommerce Website

A modern full-stack ecommerce platform built using React.js, Node.js, Express.js, and MongoDB.

This project includes:
- Customer shopping experience
- Vendor dashboard
- Admin dashboard
- Product management
- Razorpay payment integration
- Wishlist system
- Order tracking
- Authentication & authorization
- Analytics dashboard

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Jaswanth-m25/Fashion-Ecommerce.git
```

---

## 2️⃣ Backend Setup

```bash
cd backend
npm install
```

Create `.env` file inside backend:

```env
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
RAZORPAY_KEY_ID=your_razorpay_key
RAZORPAY_KEY_SECRET=your_razorpay_secret
```

Run backend:

```bash
npm start
```

---

## 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm start
```
